2. Create GoldStarAnimation: Star icon (Icons.star or custom path). AnimatedScale: 0→1.2→1.0 with Curves.elasticOut over 300ms. Color: #FBBF24 gold.

3. Create WavingMascot: Robot icon or simple vector graphic. AnimatedBuilder with rotation animation: -10° to 10° continuously over 1s. Positioned centrally.

4. Create CelebrationText: "Amazing! 🌟" in 32pt Bold purple. SlideTransition + FadeTransition (slide up 20px over 300ms).

5. Create BonusPointsDisplay: Pill Container with yellow bg. Text: "+50 Bonus Points!". ScaleTransition from 0→1 with bounce.

6. Build KidsRewardScreen (mascot): Stack → ConfettiAnimation (fills screen) → SafeArea Column (GoldStarAnimation → WavingMascot → CelebrationText → BonusPointsDisplay → "Back to Dashboard" ElevatedButton).

7. Build KidsRewardSoundScreen: Same as above without WavingMascot. Add AudioPlayer (just_audio) that plays bundled fanfare sound on screen mount.

8. Create RewardNotifier: Watches KidsModeState.quests. When all isCompleted=true, sets rewardTriggered=true. Notifier auto-navigates to /kids/reward. Awards 50 bonus points via KidsModeNotifier.

9. Add bundled audio asset: Cheerful fanfare clip (find royalty-free, or create placeholder). Add to pubspec.yaml assets.
