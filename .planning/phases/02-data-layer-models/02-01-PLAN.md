**medicine_type.dart** — The 5 medication timing types from DATA-05. Each value carries its database string:

```dart
import 'package:json_annotation/json_annotation.dart';

/// The 5 medication timing types that determine when a reminder fires
/// relative to meal anchors or other events.
///
/// Database storage: snake_case string via [dbValue].
/// JSON serialization: snake_case string via [@JsonValue].
enum MedicineType {
  /// Fire 30 minutes BEFORE the anchor meal time.
  @JsonValue('before_meal')
  beforeMeal('before_meal'),

  /// Fire 30 minutes AFTER the anchor meal time.
  @JsonValue('after_meal')
  afterMeal('after_meal'),

  /// Fire at anchor time; pre-condition: no meal in last N hours.
  @JsonValue('empty_stomach')
  emptyStomach('empty_stomach'),

  /// Fire at an absolute clock time (no anchor dependency).
  @JsonValue('fixed_time')
  fixedTime('fixed_time'),

  /// Fire N hours after previous dose was confirmed.
  @JsonValue('dose_gap')
  doseGap('dose_gap');

  const MedicineType(this.dbValue);

  /// The string stored in the `medicine_type` TEXT column in the database.
  final String dbValue;

  /// Parse from database TEXT column value.
  /// Throws [ArgumentError] if [value] doesn't match any known type.
  static MedicineType fromDbString(String value) => values.firstWhere(
        (e) => e.dbValue == value,
        orElse: () => throw ArgumentError('Unknown MedicineType: $value'),
      );
}
```

**confirmation_state.dart** — The 3 confirmation states from DATA-06:

```dart
import 'package:json_annotation/json_annotation.dart';

/// The 3 possible states of a reminder confirmation.
///
/// - [done] — User took the medication → fires downstream chain edges.
/// - [snoozed] — User deferred → same reminder re-fires after snooze_until.
/// - [skipped] — User declined → suspends all transitive downstream nodes.
///
/// Database storage: lowercase name string via [dbValue].
/// JSON serialization: lowercase string via [@JsonValue].
enum ConfirmationState {
  /// Medication taken. Triggers downstream chain nodes.
  @JsonValue('done')
  done('done'),

  /// Medication deferred. Re-fires at snooze_until time.
  @JsonValue('snoozed')
  snoozed('snoozed'),

  /// Medication declined. Suspends all downstream chain nodes.
  @JsonValue('skipped')
  skipped('skipped');

  const ConfirmationState(this.dbValue);

  /// The string stored in the `state` TEXT column in the database.
  final String dbValue;

  /// Parse from database TEXT column value.
  /// Throws [ArgumentError] if [value] doesn't match any known state.
  static ConfirmationState fromDbString(String value) => values.firstWhere(
        (e) => e.dbValue == value,
        orElse: () =>
            throw ArgumentError('Unknown ConfirmationState: $value'),
      );
}
```

**IMPORTANT:** The `@JsonValue` annotations are from `package:json_annotation`. This package is already in pubspec.yaml (Phase 01). The annotations ensure JSON serialization matches the DB string format for consistency.

**reminder_chain.dart** — Maps to `reminder_chains` table:

```dart
import 'package:freezed_annotation/freezed_annotation.dart';

part 'reminder_chain.freezed.dart';
part 'reminder_chain.g.dart';

/// A named group of reminders forming a directed acyclic graph (DAG).
/// Each chain has edges connecting reminders in a dependency order.
@freezed
abstract class ReminderChain with _$ReminderChain {
  const factory ReminderChain({
    required int id,
    required String name,
    @Default(true) bool isActive,
    required DateTime createdAt,
  }) = _ReminderChain;

  factory ReminderChain.fromJson(Map<String, dynamic> json) =>
      _$ReminderChainFromJson(json);
}
```
