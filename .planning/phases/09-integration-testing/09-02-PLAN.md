```
Scenario: Skip insulin (node 1) → verify downstream nodes are suspended
```

5. Open app → verify home screen shows skipped status for node 1, suspended for downstream

- No downstream notification fires after skip
- Home screen shows SKIPPED status for the skipped node

```

```

1. Wait for reminder notification
2. Tap SNOOZE — notification reappears after snooze interval
3. Tap SNOOZE again — 2nd snooze

6. Open app → verify home screen shows auto-skipped status

Assertions:
- After 3 snoozes, reminder transitions to SKIPPED automatically
- Auto-skip reason is recorded
- No 4th snooze notification fires (or it fires and immediately resolves to SKIPPED)
- Downstream chain nodes are suspended

**Test 4: Missed reminder detection on app open**
```
Scenario: Schedule reminder in past → open app → missed reminders surface
```

Steps:
1. Seed a reminder with scheduled_time < now and status = PENDING
2. Open the app
3. Verify missed reminder dialog/banner appears
4. Bulk mark as DONE or SKIP
5. Verify home screen updates accordingly

Assertions:
- Missed reminders are detected and surfaced to the user
- User can act on missed reminders (DONE/SKIP)
- Chain propagation works correctly after acting on missed reminders
