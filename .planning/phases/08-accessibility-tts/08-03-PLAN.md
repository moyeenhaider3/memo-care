3. **Contrast (A11Y-03):** Do all text/background color combinations pass 7:1? Update theme if default Material 3 colors fail.

Output: All screens updated + theme color overrides + golden tests.

   - Wrap in SizedBox(height: 56) or set minimumSize in ButtonStyle

   - Set constraints: `BoxConstraints(minWidth: 56, minHeight: 56)`

   - Parent SizedBox must be ≥ 56dp in both dimensions

   - Material widgets default to 48dp — increase via Transform.scale(scale: 1.2) or custom SizedBox wrapper
