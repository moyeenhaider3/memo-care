<objective>
Build the 10-second undo bar for accidental medication confirmations (A11Y-07, P-UX-02).

Elderly users with motor control issues can accidentally tap "I Took It" when they meant to skip, or vice versa. A 3-second SnackBar is too short — by the time they read it, it's gone. The undo bar persists for 10 seconds with a visible countdown and large UNDO button.

The undo flow reverses the entire confirmation cascade:
1. Delete the confirmation record
2. If DONE was undone: cancel any downstream alarms that were activated, deactivate downstream reminders
3. If SKIP was undone: reactivate any downstream reminders that were suspended, reschedule their alarms
4. Invalidate Riverpod providers so UI updates

The bar is integrated into HomeScreen (hero card actions + individual missed reminder actions). Bulk actions (Mark All Done / Skip All) intentionally do NOT get undo bars — too complex and the user made a deliberate bulk action.

Output: 3 new files + 3 updated files + 1 test file.
</objective>

**ConfirmationOutcome (from Phase 04-04):**

The ConfirmationService.confirm() returns a ConfirmationOutcome with:
- confirmationId: the ID of the saved confirmation record
- activatedReminders: List<Reminder> that were activated (for DONE)
- suspendedReminders: List<Reminder> that were suspended (for SKIP)
- state: the original ConfirmationState

This outcome is everything UndoConfirmationService needs to reverse the action.

import 'package:flutter/material.dart';

/// A 10-second undo bar shown after medication confirmation (A11Y-07).
///
/// Pinned to the bottom of the screen (above BottomNavigationBar).
/// Shows confirmed medication name, a countdown progress bar, and a
/// large UNDO button (56dp+).
///
/// Auto-dismisses after 10 seconds. If the user taps UNDO, [onUndo]
/// fires and the bar dismisses immediately.
///
/// Only one undo bar should be visible at a time. If a new confirmation
/// occurs while a previous bar is showing, replace it (parent manages state).
class UndoConfirmationBar extends StatefulWidget {
  const UndoConfirmationBar({
    required this.medicineName,
    required this.onUndo,
    required this.onDismissed,
    super.key,
  });

  /// Medicine name shown in "Confirmed [name]" text.
  final String medicineName;

  /// Called when user taps UNDO before timeout.
  final VoidCallback onUndo;

  /// Called when the 10-second timeout expires (confirmation is now permanent).
  final VoidCallback onDismissed;

  @override
  State<UndoConfirmationBar> createState() => _UndoConfirmationBarState();
}

class _UndoConfirmationBarState extends State<UndoConfirmationBar>
    with SingleTickerProviderStateMixin {
  static const _duration = Duration(seconds: 10);

  late final AnimationController _controller;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(vsync: this, duration: _duration)
      ..forward()
      ..addStatusListener((status) {
        if (status == AnimationStatus.completed) {
          widget.onDismissed();
        }
      });
  }

  @override
  void didUpdateWidget(UndoConfirmationBar oldWidget) {
    super.didUpdateWidget(oldWidget);
    // New confirmation replaces previous — restart timer
    if (oldWidget.medicineName != widget.medicineName) {
      _controller.reset();
      _controller.forward();
    }
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    final theme = Theme.of(context);
    final colorScheme = theme.colorScheme;

    return Semantics(
      liveRegion: true,
      label: 'Confirmed ${widget.medicineName}. '
          'Tap undo to reverse.',
      child: Material(
        elevation: 8,
        color: colorScheme.inverseSurface,
        child: SafeArea(
          top: false,
          child: Column(
            mainAxisSize: MainAxisSize.min,
            children: [
              // Countdown progress bar
              AnimatedBuilder(
                animation: _controller,
                builder: (context, _) => LinearProgressIndicator(
                  value: 1.0 - _controller.value,
                  backgroundColor: colorScheme.onInverseSurface.withOpacity(0.2),
                  valueColor: AlwaysStoppedAnimation(colorScheme.primary),
                  minHeight: 4,
                ),
              ),

              Padding(
                padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 12),
                child: Row(
                  children: [
                    Icon(
                      Icons.check_circle,
                      color: colorScheme.onInverseSurface,
                      size: 24,
                    ),
                    const SizedBox(width: 12),

                    // Confirmed text
                    Expanded(
                      child: Text(
                        'Confirmed ${widget.medicineName}',
                        style: theme.textTheme.bodyLarge?.copyWith(
                          color: colorScheme.onInverseSurface,
                          fontWeight: FontWeight.w500,
                        ),
                        overflow: TextOverflow.ellipsis,
                        maxLines: 1,
                      ),
                    ),

                    const SizedBox(width: 12),

                    // UNDO button — 56dp min height
                    SizedBox(
                      height: 56,
                      child: OutlinedButton(
                        onPressed: widget.onUndo,
                        style: OutlinedButton.styleFrom(
                          side: BorderSide(
                            color: colorScheme.error,
                            width: 2,
                          ),
                          padding: const EdgeInsets.symmetric(horizontal: 24),
                        ),
                        child: Text(
                          'UNDO',
                          style: theme.textTheme.titleMedium?.copyWith(
                            color: colorScheme.error,
                            fontWeight: FontWeight.bold,
                          ),
                        ),
                      ),
                    ),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

import 'package:freezed_annotation/freezed_annotation.dart';

import 'package:memo_care/core/platform/alarm_scheduler.dart';
import 'package:memo_care/features/confirmation/data/confirmation_repository.dart';
import 'package:memo_care/features/confirmation/domain/confirmation_service.dart';
import 'package:memo_care/features/reminders/data/reminder_repository.dart';
import 'package:memo_care/features/reminders/domain/models/reminder.dart';

part 'undo_confirmation_service.freezed.dart';

/// Result of an undo operation.
@freezed
class UndoResult with _$UndoResult {
  const factory UndoResult.success({required String message}) = UndoSuccess;
  const factory UndoResult.failure({required String message}) = UndoFailure;
}

/// Reverts a medication confirmation and all its side effects.
///
/// When a DONE confirmation is undone:
/// 1. Delete the confirmation record
/// 2. Cancel alarms for all downstream reminders that were activated
/// 3. Deactivate those downstream reminders
/// 4. Re-activate the original reminder
///
/// When a SKIPPED confirmation is undone:
/// 1. Delete the confirmation record
/// 2. Re-activate all downstream reminders that were suspended
/// 3. Re-schedule alarms for those downstream reminders
/// 4. Re-activate the original reminder
class UndoConfirmationService {
  UndoConfirmationService({
    required this.confirmationRepository,
    required this.reminderRepository,
    required this.alarmScheduler,
  });

  final ConfirmationRepository confirmationRepository;
  final ReminderRepository reminderRepository;
  final AlarmScheduler alarmScheduler;

  /// Reverts the confirmation described by [outcome].
  ///
  /// [outcome] is the ConfirmationOutcome returned by ConfirmationService
  /// when the confirmation was originally recorded.
  Future<UndoResult> undo(ConfirmationOutcome outcome) async {
    try {
      // 1. Delete the confirmation record
      await confirmationRepository.delete(outcome.confirmationId);

      // 2. Handle side effects based on original state
      switch (outcome.state) {
        case ConfirmationState.done:
          // Cancel alarms for activated downstream reminders
          for (final reminder in outcome.activatedReminders) {
            await alarmScheduler.cancel(reminder.id);
            await reminderRepository.deactivate(reminder.id);
          }

        case ConfirmationState.skipped:
          // Reactivate and reschedule suspended downstream reminders
          for (final reminder in outcome.suspendedReminders) {
            await reminderRepository.activate(reminder.id);
            await alarmScheduler.schedule(reminder);
          }

        case ConfirmationState.snoozed:
          // Snoozed: cancel the rescheduled snooze alarm
          await alarmScheduler.cancel(outcome.reminderId);
      }

      // 3. Restore original reminder to active/pending
      await reminderRepository.activate(outcome.reminderId);

      return UndoResult.success(
        message: 'Undone — ${outcome.medicineName ?? "medication"} '
            'marked as pending',
      );
    } catch (e) {
      return UndoResult.failure(
        message: 'Could not undo: $e',
      );
    }
  }
}
