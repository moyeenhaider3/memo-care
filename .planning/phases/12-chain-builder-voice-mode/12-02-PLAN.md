**12-01 provides:** Chain builder screen with canvas, nodes, connectors, add step. All static positioning.

**This plan adds interactive features:**

- Drag-and-drop nodes on canvas with snap-to-grid
- Mini-map for navigation (128×160px, bottom-right)
- Zoom controls (+/-/center)
- Timeline simulation (bottom sheet, fast-forward chain execution)
- Save chain (validates + persists)
- Test run (simulates without saving)
  </context>

<steps>
1. Create DraggableNodeWrapper: Wraps any node widget. GestureDetector with onPanStart/Update/End. On drag: update node position in notifier. On release: snap to grid (round to 24px). Scale feedback during drag.

2. Wrap all AnchorNode and ChainNode in DraggableNodeWrapper. Connector lines auto-update via state rebuild.

3. Create MiniMap: Container(128×160) with CustomPainter. Draws simplified colored rectangles for each node at proportional positions. Semi-transparent viewport indicator rectangle. GestureDetector tap → calculate canvas offset → jump.

4. Create ZoomControls: Column of 3 FloatingActionButton.small widgets. Wire to InteractiveViewer TransformationController: scaleBy(1.2), scaleBy(0.8), resetToFit().

5. Create SimulationState (Freezed): steps (List<SimulationStep>), currentIndex, isRunning, isPaused, speed.

6. Create SimulationNotifier: Takes chain nodes → generates ordered execution steps → Timer.periodic advances through steps.

7. Create TimelineSimulationSheet: ModalBottomSheet with list of simulation steps. Current step highlighted. Colored dot per step. Play/Pause button. Speed control.

8. Create SaveChainButton: AppBar action. Converts BuilderNodes to ChainData. Calls ChainValidator → show errors or save via ChainRepository. Success snackbar.

9. Create TestRunButton: AppBar action. Opens TimelineSimulationSheet without saving.

10. Update ChainBuilderScreen: Add MiniMap (Positioned bottom-right), ZoomControls (above mini-map), AppBar actions (Save + Test Run). Wire InteractiveViewer.

11. Widget tests: Drag node updates position, snap to grid works, mini-map renders, zoom scales, save validates.
    </steps>
