<steps>
1. Create BuilderNode (Freezed): id (String uuid), type (NodeType enum: anchor/medicine/meal/activity), name, dosage, timeOffset, relationship (afterDone/always), position (Offset), parentId.

2. Create ChainBuilderState (Freezed): nodes (List<BuilderNode>), selectedNodeId, canvasOffset (Offset), zoomLevel (double, default 1.0).

3. Create ChainBuilderNotifier: Methods — addNode (creates node at calculated position below parent), removeNode, updateNode, selectNode, updateNodePosition, saveChain (converts to ChainEngine input and validates).

4. Create DotGridCanvas: CustomPainter that draws circles (2px radius) at 24px grid intervals. Color: grey at 0.2 opacity. Paint in visible viewport only for performance.

5. Create AnchorNode: Container with pill shape (999 radius), gold bg, white border. Row(schedule icon + time text). InkWell → TimePicker. Positioned widget.

6. Create ChainNode: SizedBox(width: 320). Card with left colored strip (Container). Content: name, dosage, offset text, edit icon button. Offset badge (Positioned top-right). Tap → select. Long press → edit dialog.

7. Create NodeConnector: CustomPainter that draws path from parent bottom-center to child top-center. Solid stroke for afterDone, dashPath for always. Small label pill at midpoint.

8. Create AddStepButton: Container with dashed border (DashDecoration or custom paint), accent blue bg at 10%. Icon + text. Tap → showDialog(AddStepDialog) with name, type picker, relationship dropdown.

9. Build ChainBuilderScreen: Scaffold → AppBar ("Chain Builder", actions placeholder) → Stack(InteractiveViewer → CustomMultiChildLayout or Stack with Positioned children: DotGridCanvas + AnchorNode + ChainNodes + Connectors + AddStepButton).

10. Update AppRouter with /chain-builder.

11. Widget tests: Verify canvas paints grid, nodes render, add step creates new node, connectors draw between nodes.
    </steps>
