**AndroidManifest.xml** — Add all 10 required permissions BEFORE the `<application>` tag. Keep all existing content (application, activity, meta-data, intent-filter) unchanged.

Add these permissions with comments explaining each:
```xml
<!-- Exact alarms (Android 12+) — medication timing must not drift -->
<uses-permission android:name="android.permission.SCHEDULE_EXACT_ALARM" />
<!-- Android 14+ health/medication apps use USE_EXACT_ALARM -->
<uses-permission android:name="android.permission.USE_EXACT_ALARM" />

<!-- Notifications (Android 13+ requires runtime permission) -->
<uses-permission android:name="android.permission.POST_NOTIFICATIONS" />

<!-- Full-screen intent for escalation takeover (tier 3) -->
<uses-permission android:name="android.permission.USE_FULL_SCREEN_INTENT" />
<!-- System alert window for overlay if full-screen denied -->
<uses-permission android:name="android.permission.SYSTEM_ALERT_WINDOW" />

<!-- Wake device for critical alerts -->
<uses-permission android:name="android.permission.WAKE_LOCK" />

<!-- Reschedule alarms after device restart -->
<uses-permission android:name="android.permission.RECEIVE_BOOT_COMPLETED" />

<!-- Microphone for voice commands (Hindi + English) -->
<uses-permission android:name="android.permission.RECORD_AUDIO" />

<!-- Vibration for escalation tier 2 -->
<uses-permission android:name="android.permission.VIBRATE" />

<!-- Foreground service for alarm audio playback -->
<uses-permission android:name="android.permission.FOREGROUND_SERVICE" />
```

**build.gradle.kts** — Set explicit minSdk and targetSdk values. Replace `flutter.minSdkVersion` with `24` and `flutter.targetSdkVersion` with `35`.

```kotlin
defaultConfig {
    applicationId = "com.example.memo_care"
    minSdk = 24       // Drift minimum, Android 7.0
    targetSdk = 35    // Play Store 2026 compliance, Android 15
    versionCode = flutter.versionCode
    versionName = flutter.versionName
}
```

Also add `multiDexEnabled = true` to defaultConfig (33 packages may exceed the 64K method limit).

Also add `coreLibraryDesugaring` to support Java 8+ APIs on API 24-25:
```kotlin
compileOptions {
    isCoreLibraryDesugaringEnabled = true
    sourceCompatibility = JavaVersion.VERSION_17
    targetCompatibility = JavaVersion.VERSION_17
}

dependencies {
    coreLibraryDesugaring("com.android.tools:desugar_jdk_libs:2.1.4")
}
```

**WHY minSdk=24:** Drift requires API 24 minimum. It covers 99%+ of active Android devices.
**WHY targetSdk=35:** Google Play Store requires targetSdkVersion 35 for new apps in 2026.
**WHY desugaring:** API 24-25 lack some Java 8 APIs used by notification scheduling libraries.
**WHY multiDex:** 33 packages with transitive dependencies may exceed 64K DEX method limit on API 24.
