**Current settings_screen.dart:** Basic settings with snooze duration, escalation timeouts, notification preferences. No profile header, no fasting toggle, no export, no caregiver management.

**Key additions:**

- Display settings (text size slider, contrast toggle, dark mode toggle)

<steps>
1. Create UserProfile domain model (Freezed): name, profileType (elderly/adult/parent), condition, avatarPath. Store via existing settings persistence.

2. Create ProfileHeader widget: CircleAvatar (72px) + name + badge + edit button.

3. Create DisplaySettingsSection: Text Size slider (maps to font scale factor), High Contrast Switch, Dark Mode Switch. Persist via SettingsNotifier.

4. Create FastingToggleTile: Gold-accent moon icon + Switch. Persists fasting mode state.

5. Create CaregiverSection: Shows linked caregiver or "Add" prompt. Phone input with send test alert/remove actions.

6. Create DataExportSection: PDF and CSV export buttons. Wire to history data → generate file → share sheet.

7. Rebuild SettingsScreen: ProfileHeader → sections with dividers → AppInfo at bottom. All using new design tokens.

8. Update SettingsNotifier: Add profile, display, fasting, caregiver state management.

9. Widget tests: Verify sections render, toggles work, profile displays.
   </steps>
