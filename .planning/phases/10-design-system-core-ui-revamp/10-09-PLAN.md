**Current onboarding:** GoRouter ShellRoute with 6 sub-routes (condition → template → anchors → medicines → review → permissions). Linear push navigation. No welcome splash, no profile type, no accessibility, no caregiver link, no celebration.

**Merged flow adds:**

1. Welcome splash with gradient hero
2. Profile type selection (Elderly/Adult/Parent)
3. Accessibility settings toggle
4. Caregiver phone link
5. Celebration with confetti

**Architecture change:** Replace GoRouter sub-routes with PageView for smooth swiping + dot indicators. OnboardingFlow becomes a single route that manages internal page navigation.
</context>
